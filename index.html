Google Refresh Token 生成器

0 前置准备

  • 前往 Google Cloud Console 创建项目
  • 启用 Google Play Android Developer API
  • 创建 OAuth 2.0 客户端 ID，应用类型选择 "Web应用"
  • 获取 client_id、client_secret 和已获授权的重定向 URI
  • 将应用发布状态从测试改为正式：应用发布状态

1 填写配置

客户端密钥，仅用于本地换取 token，不会上传至任何服务器
Google Cloud Console 中已获授权的重定向 URI，需与控制台配置一致
固定为 Google Play Android Developer API 所需的 Scope

2 授权

点击下方按钮，在浏览器中使用公司主体创建的Google账号登录并授权：

授权链接： 打开链接
授权后，浏览器会跳转到你配置的重定向 URI（页面可能报错，这是正常的），请从地址栏复制完整的 URL

3 换取 Refresh Token

将授权后浏览器地址栏的完整 URL 粘贴到下方：
